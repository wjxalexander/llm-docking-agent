# Change: Add Project Structure Constraint

## Why
To ensure stability and predictability of the project layout, it is necessary to restrict unauthorized or frequent changes to the project directory structure.

## What Changes
- Added a specific constraint to `openspec/project.md` regarding project directory modifications.
- Defined that any structural changes require a formal proposal and explicit justification.

## Impact
- Affected files: `openspec/project.md`
- AI assistants will be more cautious about creating, moving, or deleting directories.
